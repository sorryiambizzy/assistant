# Краткое содержание

Обсуждение потенциала продукта для криптообменников на платформе SMRM. Анализ рынка, конкурентов, инвестиционной привлекательности.

# Основные темы

## 1. Анализ рынка криптообменников

### Текущая ситуация
- ~200 точек (криптообменников) на рынке
- С появлением регуляции количество точек сократится, произойдёт консолидация
- Мелкие игроки не учитываются
- Достижимая доля рынка: ~30%

### Финансовая оценка
- Потенциальная выручка: ~$216,000 в год
- Рынок оценивается как «маленький» — недостаточная привлекательность для инвестиций

## 2. Конкурентная среда

### Текущие конкуренты
- **СЕРМ система** (премиум-сегмент)
- **Best**
- **Exchange**
- **Сеть "7"** — оффлайн офисы
- **Банки** — потенциально быстро войдут на рынок с отдельными модулями

### Инсайт о банках
Банки рассматривают криптообмен как аналог SWIFT для крупных безналичных расчётов (покупки за рубежом, автомобили). С переходом рынка в белую зону банки станут активными конкурентами.

## 3. Текущая оценка SMRM

### Необходимые доработки
- Реализация
- Настройка
- Техническая платформа
- Внедрение (все хотят кастом)

### Проблемы
- Нет продукта с предметной экспертизой в крипте
- Нет повторяемых паттернов — каждый обменник уникален
- Тяжёлый конфигурационный продукт → мелкие истории покрывают скрипты + боты

## 4. Тренды рынка

### Негативные факторы
- Объём обмена крипты уменьшается на фоне экономической ситуации
- Один из собеседников (Белок) сообщил о падении бизнеса на 30% из-за блокировок по 115-ФЗ

### Позитивные факторы
- Рынок крипты вырастет с переходом в белую зону (Газпром, Лукойл и др.)
- Но этот объём недоступен для SMRM — другие масштабы

## 5. Риски и ограничения

### Выборка собеседников
- Видели только «белую» и «серую» зоны рынка (через Игната)
- Не слышали мнения тех, кто не работает с регулятором

### Серые зоны
Даже после принятия закона останутся большие серые зоны — не все будут легализовываться.

## 6. Возможные действия

### Вариант 1: Найти продакта-эксперта
- Найти человека с опытом в крипте/финтехе
- Например: бывший сотрудник FastTisi (октябрь 2024 — февраль 2026)
- **Действие**: [[10_PEOPLE/epodkin/epodkin.md|Евгений]] → найти контакт, поставить встречу

### Вариант 2: Конфигурация SMRM
- Дождаться оценки от Муратовой (ITSM365)
- Рассмотреть отдельную конфигурацию «Крипто»
- **Ограничение**: нужен product, понимающий рынок

### Вариант 3: Совместное предприятие
- Найти местного игрока на рынке криптообменников
- Создать JV 50/50: софт от Naumen + экспертиzata партнёра
- Форкнуть платформу в отдельную структуру
- **Идеальный сценарий при неограниченных ресурсах**
