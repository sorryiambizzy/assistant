## Ключевые темы

### Грамокс — основные преимущества

- **Independence:** Если нужно, Грамокс можно удалить — не все решения зависят от него
- **Flexibility:** В отличие от Эмбис, где все решения завязаны на систему
- **Integration:** Работает с GitLab — группы, проекты, доступы

### Персональные ассистенты vs Проекты

| Персональный ассистент | Проект |
|------------------------|--------|
| Для личной работы | Для командной работы |
| Не делится с другими | Общедоступный |
| Операционные задачи | Проектная деятельность |
| Пример: управленец с задачами | Пример: продукт с командой |

**Ключевой принцип:**
- Персональный ассистент → личная организация, не для публикации
- Проект → командная работа, документация, задачи, календарные планы

### Шаблоны проектов

В Грамоксе уже есть `project template` с базовой заготовкой для документации.

**Текущие шаблоны:**
- Персональные ассистенты
- Шаблон для разработки (встройка, Cruzi модуль)

**Запрошенные шаблоны:**
- Шаблон проекта для документации
- Другие специализированные шаблоны (запрос к участникам)
