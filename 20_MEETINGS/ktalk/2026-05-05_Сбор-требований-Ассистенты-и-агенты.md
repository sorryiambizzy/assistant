## Ключевые темы

### Подход Игоря Щепочкина

**Роли:** Рассуждатор + Исследователь

**Вывод:**
> *"Коллизация через VS Code, через консоль, через Cursor — это просто увеличение контекста с персистентным хранением. В файловой системе твой файлик будет всё расти, расти и расти."*

Для его объёма контекста достаточно **2 ролей**, не нужна сложная система.

### Требования Артура Чернякина

**Нужно:**
1. **Вытаскивать кодовую базу по модулям** в общее пространство
2. **Раскладывать по индексированным файлам** с необходимым контекстом
3. **Персонализированные агенты** по каждому модулю
4. **Доступ к документации** — чтобы агент не "чудил"
5. **Генерировать концепты** для разработчиков

**Применение:**
- Ответы на Asks
- Подготовка аналитики
- Генерация валидных решений для конкретного модуля
- Переиспользование компонентов из кода

> *"Хочу, чтобы на выходе можно было ставить задачу, и он уже концепт выдавал максимально близкий к тому, что разработчик может взять, провалидировать, причесать и пушнуть."*

### Архитектура решения (Максим Демьянов)

**Два ассистента:**

| Ассистент | Назначение | Частота работы |
|-----------|------------|----------------|
| **Персональный менеджер** | Агрегировать всю деятельность, решать задачи менеджера | 3 раза в день (утро, обед, вечер) + 2 раза в неделю (план/ретроспектива) |
| **Проектный ассистент** | Большие проекты, работа с кодовой базой | По необходимости |

**Важно:** Сначала — себе (как в самолёте), потом — другим.

## Структура ассистентов

### Агенты (Agents)

Находятся в папке `.agents`:
- **Аналитик** — анализ
- **Ревью** — review
- **Бизнес-аналитик** — бизнес-анализ
- **Создатель документации** — документация

> *"Помните Матрицу? Там был Agent Smith. Вот, у нас примерно то же самое."*

### Команды (Commands)

Небольшие инструкции, которые не нужно упаковывать в агента.

**Примеры:**
- `vk review` — посмотреть код

## Решения и договорённости

### Согласованные подходы

1. **Пилот:** Берём за основу ассистента из пилота
2. **Два ассистента:** Персональный + Проектный
3. **Агенты:** Виртуальные, легко нанимать и увольнять
4. **Приоритет:** Сначала — себе, потом — другим
