## Ключевые решения

### 1. Грамокс — выбранный инструмент

- Подтверждено: пилот делаем именно на Грамоксе, без откатывания назад
- Это пилот в формате: тестируем инструмент → принимаем решение
- Параллельно — пилот формата взаимодействия "AI Adoption ↔ ТП"

### 2. Версия SMP — 4.21.0

- Костя выбрал — на эту версию они **обновляют всех клиентов сейчас**
- Будет востребована = подходит для пилота

### 3. Формат теста

- ТП тестирует **встроенный поиск Грамокса** (агентов у поддержки нет)
- Параллельно AI Adoption тестирует через ассистентов (Антон, Даша)
- Если по итогам Грамокс хорош только для ассистентов, не для поиска — отдельный трек подключения поддержки к ассистентам

### 4. Курс Муратова

- Можно адаптировать под ТП — если решат вкатываться в ассистенты после пилота
- ЕП пришлёт ссылку
