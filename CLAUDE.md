## Инструменты

| Инструмент | Когда использовать |
|------------|-------------------|
| **qmd** | Семантический поиск по vault (MCP) |
| **Filesystem** | Чтение, создание, редактирование файлов |
| **Web Search** | Поиск актуальной информации |

**qmd — быстрый старт:**
- `qmd_deep_search` — основной поиск (гибридный с reranking)
- `qmd_search` — точный поиск по ID, frontmatter
- `qmd_get` — получить конкретный файл

**Золотое правило:** Поиск по ID (`epodkin`) работает 100%. По русским именам — используй `qmd_deep_search`.

> Если qmd не установлен — используй встроенные Grep/Glob/Read.

Подробнее: `.claude/docs/qmd-reference.md`
